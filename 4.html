농부의 WEB HOSTING 정리


새로운 HTML파일을 업로드하고 호스팅하는 방법

  • 1. github.com 으로 접속한다.

  • 2. 프로필 왼쪽 +(add)버튼을 클릭하 new repository를 클릭한다.

  • 3. 프로젝트 파일명과 repository name을 같게 설정하고 add a readme file을 체크한후 creat repository를 클릭한다

  • 4. 그 후 setting으로 들어가서 첫화면에서 스크롤을 내리면 github pages라는 항목이 나오는데 source 항목에서 branch를 main으로 선택하고 save를 눌러준다.

  • 5. 그러면 github pages에서 등록된 나의 html의 주소를 보여준다.

기존에 업로드 된 파일을 업데이트 하는 방법

  • 1. 프로필을 클릭한다.

  • 2. your repository를 클릭한다.

  • 3. add file을 클릭하고 update files를 클릭한다.

  • 4. HTML 폴더의 자료들을 전부 드래그하여 업데이트를 완료한다.
